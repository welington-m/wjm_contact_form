Um plugin de formulário de contato simples, seguro e altamente personalizável, baseado em princípios de arquitetura limpa.

== Description ==

**WJM Contact Form** é um plugin de contato desenvolvido com boas práticas modernas como Clean Architecture e DDD. Ele permite:

- Campos de formulário personalizáveis via painel
- Validação do lado do servidor
- Registro de mensagens no banco de dados
- Exportação em CSV e JSON
- Painel administrativo com filtro, busca e paginação
- Totalmente adaptável a múltiplos idiomas

Ideal para desenvolvedores que querem um plugin leve, extensível e seguro.

== Installation ==

1. Faça o upload da pasta `wjm-contact-form` para o diretório `/wp-content/plugins/`
2. Ative o plugin pelo menu “Plugins” no WordPress
3. Acesse “Configurações > Formulário WJM” para configurar os campos
4. Use o shortcode `[wjm_contact_form]` em qualquer página ou post

== Frequently Asked Questions ==

= Posso mudar os textos dos campos? =
Sim! Todos os rótulos e mensagens podem ser configurados pelo painel.

= O plugin salva os dados no banco de dados? =
Sim, e você pode visualizá-los no menu “Mensagens WJM” no painel.

= Posso exportar os dados? =
Sim. Os dados podem ser exportados em formato CSV (mensagens) e JSON (configurações).
